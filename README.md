# homebridge-docker-compose

## Run with portainer
1. install portainer community edition
2. add a git stack for each module
3. use .env.example for env variables in stack definition

## Services:
- Utility services
  - Wireguard
  - Uptime-Kuma
- Media-management:
  - Sonarr
  - Radarr
  - Bazarr
  - Prowlarr
  - Jellyfin
  - Deluge
- Home automation
  - Homebridge
- ScPrime
